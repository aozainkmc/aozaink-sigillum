# aozaink-sigillum 当前实现说明

本文件只记录当前代码已经实现的行为。玩法计划、未来联动和未落地想法放在 `DESIGN.md`。

## 总体边界

- 当前普通黄符 UI 语义为 **两个咒位 + 一个尾修**：前两格可写技能字或修饰字，第三格只接受修饰字或留空；新符不再产生三技能字组合。
- 当前组合符已支持 12 个技能字的 **66 对二字特殊联动**；二技能带一个尾修时继续走对应二字联动，并把尾修作为联动整体的外层策略。单技能带双修饰时仍按同一张符的整体品质和修饰字通用规则结算。
- 二字联动通过无序 exact pair registry 解析：每一对技能字只注册一个 `ComboSpec`，不存在运行时抢解释权；重复注册会在初始化时报错。
- 两个咒位不区分主副。代码按物品里保存的槽位顺序遍历技能，但不定义“主技能/副技能”。
- 重复玩法字判废符；没有技能字的普通符判废符。
- `广` 和 `穿` 不能同时出现，同时出现判废符。
- 单技能纯自施（护/净/魄）不能使用 `广` 或 `穿`，判废符。二技能带 `广` 时，若至少包含 `护`/`净` 这类支援资源，则可作为广域支援生效。`明` 同时具备自施夜视和目标显形效果，因此可使用 `广` 或 `穿`。
- 二技能带 `穿` 时按 linked combo 的路径语义决定是否支持；不支持的组合会消耗并提示不支持穿透。

## 品质倍率

| 品质 | 阈值 | 倍率 |
|---|---:|---:|
| 极品 | >= 0.95 | 1.00 |
| 良品 | >= 0.75 | 0.80 |
| 劣质 | >= 0.50 | 0.55 |
| 废符 | < 0.50 | 0.00 |

组合符整体倍率取三格中最差品质。空格不参与最差品质计算。

## 直接施法

| 字 | 当前代码行为 |
|---|---|
| 镇 | 目标减速和虚弱 60 tick；亡灵更强且时长 1.5 倍。`强` 会把控制等级提高一级，`续` 延长时长。 |
| 封 | 目标虚弱 X、挖掘疲劳 V 60 tick；亡灵额外高等级减速和跳跃锁定 90 tick。`强` 提高效果等级，`续` 延长时长。 |
| 退 | 击退目标并给 20 tick 减速 IV；亡灵击退距离翻倍。`强` 提高击退强度，`续` 延长减速。 |
| 引 | 非玩家目标被拉向施法者，拉力随距离递减。`强` 提高拉力，`续` 当前无收益。 |
| 火 | 点燃式 DoT 4 秒，普通 2 点/秒，亡灵 3 点/秒，按倍率缩放。`续` 延长燃烧秒数。 |
| 雷 | 生成原版 `LightningBolt`，设置施法者为 cause；下一 tick 追加 12 点雷击伤害，雨天 x1.3，50% 概率给 10 tick 高等级减速和虚弱。 |
| 护 | 给玩家额外护盾条：极品 30 点、良品 16 点、劣质 8 点；护盾不随时间掉，只在受击时消耗。`强` 可把护盾量乘上倍率，`续` 无收益。 |
| 净 | 清除自身负面效果：极品 3 个、良品 2 个、劣质 1 个；若视线目标是亡灵，额外造成 4 点圣化魔法伤害。`续` 当前无收益。 |
| 斩 | 目标血量比例低于阈值时以玩家攻击伤害处决：极品 50%、良品 25%、劣质 10%；`强` 后最终阈值封顶 75%。未达阈值只出烟雾。 |
| 明 | 自身夜视，基础 6000 tick；若视线命中生物，则该生物发光同等时长。未命中仍给夜视。`广` 改为 24 格范围照妖并把基础时长降为 1500 tick；`穿` 沿视线标记多个目标。`续` 延长持续时间；`强` 当前不改变该技能。 |
| 吸 | 对目标造成魔法伤害并给玩家等量回血：极品 1.5 心、良品 1 心、劣质 0.5 心；`强` 提高伤害和回血，`续` 当前无收益。 |
| 魄 | 召回上次死亡掉落；极品 100%、良品 75%、劣质 50%。`强` 可把低品质提升到 100% 档，`续` 当前无收益。 |

## 修饰字

| 字 | 当前代码行为 |
|---|---|
| 强 | 令效果倍率变为 `overallM * 2.0`。只影响读取 `CastEnv.multiplier()` 的技能。 |
| 续 | 令持续时间倍率变为 `1.0 + overallM`。只影响读取 `CastEnv.durationMultiplier()` 的技能。 |
| 广 | 普通目标技能以命中的主目标为中心取半径 4 格 AABB，对范围内活体逐个结算目标效果，效果倍率再乘 0.7；混合符中的普通自施效果仍正常对玩家生效。二技能带 `广` 时继续走二字联动：需要目标的联动对主目标周围半径 4 格目标逐个执行，主目标那次允许支援结算，其他目标压制支援以避免复制资源；包含 `护`/`净` 时触发广域支援，护盾量/净化次数先补施法者，吃不下的部分按距离从近到远给范围内其他玩家。`明+广` 例外为以施法者为中心 24 格范围照妖，基础时长 1500 tick。 |
| 穿 | 单技能穿透：极品最多 3 个目标、良品 2 个、劣质 1 个；命中后按命中次数降低对应技能格品质并返还降档后的符。二技能带 `穿` 时只有可沿射线逐目标解释的 linked combo 生效；含护盾/净化监听/魄/大多数明扫描语义的组合会消耗并提示不支持穿透。`明+穿` 命中后同时给玩家夜视。 |

## 二字技能联动

仅当普通组合符恰好有 **2 个不同技能字且无修饰字** 时启用。当前 12 选 2 共 66 对都有 `LinkedComboSpec`，由 spec 决定是否需要视线目标；`护+净`、`护+明`、`净+明` 这类没有必需目标的联动不需要瞄准实体。

| 锚点 | 配对 | 实际行为 |
|---|---|---|
| 吸 | 任意其他技能字 | 先执行另一技能，再按该技能的实际伤害/位移/状态追加吸血或回血。 |
| 魄 | 退 / 引 | 不需要瞄准实体；召回成功时分别从玩家与死亡点触发驱退脉冲（退），或把掉落直接送到脚底（引）。 |
| 魄 | 其他技能字 | 先对该技能需要的目标生效，再同时执行魄的掉落召回。 |
| 镇 | 封 / 退 / 引 / 火 / 雷 / 护 / 净 / 斩 / 明 | 镇系联动：稳定镇封、击退/牵引后镇压、镇压燃烧、雷锚链跳、移动转盾、负面抵消圣化、移动重检斩杀、死亡转移镇压。 |
| 封 | 退 / 引 / 雷 / 护 / 净 / 斩 | 封系联动：击飞落地封、牵引封、封禁期循环雷击、封伤转盾、净化圣化后封禁、封禁提高斩杀线。 |
| 封 | 火 / 明 | 直接组合：封禁与火/明按单字同时生效，不做额外联动。 |
| 退 | 引 / 火 / 雷 / 护 / 净 / 斩 / 明 | 位移联动：击退落点聚怪、落点火花、雷击冲击波、动能反推护盾、净化圣化击退、追斩、落点照妖。 |
| 引 | 火 / 雷 / 护 / 净 / 斩 / 明 | 牵引联动：燃烧牵引线、路径落雷、护盾边界牵引、牵引圣化爆发、近身牵引斩、牵引延长照妖。 |
| 火 | 雷 / 护 / 净 / 斩 / 明 | 元素联动：火雷引爆、燃烧伤害转盾、净焰、燃烧触发斩杀、火炬标记。 |
| 雷 | 护 / 净 / 斩 / 明 | 雷系联动：雷击转额外盾、净雷破增益、雷后斩杀、闪照标记。 |
| 护 | 净 / 斩 / 明 | 护系联动：净护抵消负面、处决转盾、明护反标记攻击者。 |
| 净 | 斩 / 明 | 净系联动：净化圣化斩、净明筛选标记。 |
| 斩 | 明 | 标记范围敌人，从主目标开始寻找首个低于斩杀线的目标处决。 |

所有二字联动都按无序 pair 精确查表。`吸+镇`、`吸+魄`、`镇+魄` 等重叠锚点组合也只是各自独立的 exact pair，不再用“吸/魄/镇谁优先”描述。

## 修饰组合（非二字联动）

当前没有“三技能字组合”设计；新 UI 已阻止第三格写技能字。二技能带一个尾修时仍走二字联动，尾修作为外层策略进入联动：

| 组合 | 当前实际结果 |
|---|---|
| 火 + 雷 | 恰好二技能且无修饰时触发火雷引爆。 |
| 火 + 雷 + 强 | 触发火雷引爆联动，并用 `强` 提高整体联动倍率。 |
| 镇 + 封 + 续 | 触发镇封联动，并用 `续` 延长共同控制窗口。 |
| 退 + 引 | 恰好二技能且无修饰时触发退引联动，不是两个技能各自为政。 |
| 火 + 护 + 广 | 触发燃烧伤害转盾联动；目标效果范围化，护盾先给施法者，溢出按距离给范围内其他玩家。 |
| 护 + 净 + 广 | 无目标技能也可生效：护盾量和净化次数先补施法者，溢出按距离给范围内其他玩家。 |
| 火 + 雷 + 穿 | 触发火雷引爆联动的穿透版，沿射线逐个目标结算。 |
| 护 + 净 + 穿 | 不支持穿透，使用时消耗并提示不支持。 |

## 刻印

- 刻印符前两个咒位中包含且只包含一个 `刻`；第三格仍是尾修槽，只能为空或修饰字。
- `刻 + 技能` 创建服务端持久刻印；`刻 + 技能 + 修饰` 创建带单个修饰的单技能刻印；`穿` 判废符。
- `刻 + 续/强/广` 可对已有刻印执行续时/补能、一次加强、扩大范围；没有对准已有刻印时不消耗并提示。
- `刻 + 修饰 + 修饰` 可对已有刻印连续执行两个合法操作；含 `穿` 判废符。
- `刻 + 技能 + 技能`、第三格写 `刻` 或技能字均判废符。
- 默认时间和能量为 36000 tick，剩余时间上限 24 小时。`刻 + 护` 的初始能量为普通刻印 5 倍。
- 默认半径：极品 10 格、良品 8 格、劣质 5 格；`广` 后翻倍。
- 刻印暂停跟随游戏 tick：区块未加载、服务器暂停或退出游戏时不扣剩余 tick。
